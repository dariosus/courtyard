¿De qué se trata el curso?

¡Te damos la bienvenida al curso de Desarrollo web full stack!

Sumergite en este mundo, conocé las tecnologías más novedosas y las herramientas que necesitás para poder desarrollar sun sitio web de punta a punta. A lo largo de estos 5 meses vas a trabajar en equipo, pensar proyectos, resolver desafíos, investigar y meter mucho las manos en el código.

¿Empezamos?

¿Qué vas a lograr en el curso?

En el curso de Desarrollo web Full Stack nos proponemos lograr que adquieras y desarrolles las herramientas, habilidades y conocimientos técnicos necesarios para resolver situaciones que posteriormente se le van a presentar en la vida real. También buscaremos que aprendas a “pensar como programador/a”, habilidad que incluye desde armar un sistema hasta lograr identificar errores en el mismo, proponer soluciones y actualizar en base a cambios en las tecnologías.

Buscaremos fomentar constantemente la investigación, la búsqueda de información, la lectura de documentación técnica y el trabajo en equipo.

¿Qué vas a aprender en el curso?

  • Adquirir los conocimientos necesarios para desarrollar un sitio web de punta a punta integrando front-end y back-end.
  • Identificar fallas en un proyecto e implementar las modificaciones necesarias para resolverlo.
  • Trabajar en equipo.
  • Saber a qué documentación y comunidades recurrir para resolver problemas y mantenerse actualizado en cada uno de los lenguajes aprendidos.
  • Trabajar con frameworks para poder generar proyectos escalables, seguros y performantes.

¿Qué vas a ver en el curso?

JavaScript y NODE

Vamos a conocer el potencial que nos dá este lenguaje y experimentar las diversidad de aplicaciones que se pueden realizar gracias al entorno de desarrollo de NODE.

Git y SCRUM

Aprender metodologías y herramientas de trabajo que se utilizan en el ámbito del desarrollo.

HTML + CSS

Comprender cómo se estructura un sitio web y cómo podemos darle estilo y diseño a esa estructura.

Express

Desarrollaremos el concepto de FullStack al unir el mundo del Front-End con el Backend. Para esto utilizaremos el framework más usado en el ecosistema de NODE.

SQL

Conoceremos qué son las bases de datos y cómo podemos operar sobre las mismas.

JavaScript front-end y React

Vamos a agregar dinamismo e interactividad a nuestro Front-End. Luego veremos cómo reutilizar lógica con una introducción de componentes con React.

Deploy de un proyecto

Aprender como hacer el deploy de un desarrollo y exponer el proyecto integrador

Programación orientada a objetos

Taller optativo y extracurricular para conocer sobre el paradigma de programación orientada a objetos.
